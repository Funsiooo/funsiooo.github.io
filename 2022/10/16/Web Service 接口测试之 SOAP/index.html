SOAP, жјҸжҙһеӯҰд№ 

дёҖгҖҒWeb Service жҰӮиҝ°

WebService жҳҜдёҖз§Қи·Ёе№іеҸ°пјҢи·ЁиҜӯиЁҖзҡ„и§„иҢғпјҢз”ЁдәҺдёҚеҗҢе№іеҸ°пјҢдёҚеҗҢиҜӯиЁҖејҖеҸ‘зҡ„еә”з”Ёд№Ӣй—ҙзҡ„дәӨдә’гҖӮеҚі JAVA еә”з”ЁзЁӢеәҸеҸҜд»ҘйҖҡиҝҮ websrvice и°ғз”Ё PHP жҲ–иҖ… Python зӯүзЁӢеәҸжҸҗдҫӣзҡ„жңҚеҠЎгҖӮWebService дёүиҰҒзҙ пјҡSOAPгҖҒWSDLгҖҒUDDIгҖӮ

WebService дёүиҰҒзҙ пјҡ

SOAPпјҲз®ҖеҚ•еҜ№иұЎи®ҝй—®еҚҸи®®пјү жҳҜеҹәдәҺ XML зҡ„з®Җжҳ“еҚҸи®®пјҢеҸҜдҪҝеә”з”ЁзЁӢеәҸеңЁ HTTP д№ӢдёҠиҝӣиЎҢдҝЎжҒҜдәӨжҚўпјҢжҳҜиҝһжҺҘжҲ– Web жңҚеҠЎжҲ–е®ўжҲ·з«Ҝе’Ң Web жңҚеҠЎд№Ӣй—ҙзҡ„жҺҘеҸЈгҖӮз®ҖеҚ•ең°иҜҙе°ұжҳҜпјҢSOAP жҳҜз”ЁдәҺи®ҝй—®зҪ‘з»ңжңҚеҠЎзҡ„еҚҸи®®гҖӮ

WSDLз”ЁжқҘжҸҸиҝ°еҰӮдҪ•и®ҝй—®е…·дҪ“зҡ„жҺҘеҸЈгҖӮе‘ҠиҜүи°ғз”ЁиҖ…еә”иҜҘжҖҺд№ҲеЎ«еҶҷSOAPеҪ“дёӯзҡ„XMLж–Үд»¶йҮҢйқўзҡ„еҶ…е®№гҖӮжңҚеҠЎз«Ҝе…·дҪ“жҸҗдҫӣдәҶд»Җд№ҲжңҚеҠЎпјҢиҜҘжҖҺд№Ҳи°ғз”ЁиҝҷдёӘжңҚеҠЎпјҢиҜҘдј йҖ’д»Җд№Ҳзұ»еһӢзҡ„еҸӮж•°гҖӮдјҡиҝ”еӣһд»Җд№Ҳзұ»еһӢзҡ„ж•°жҚ®зӯүзӯүгҖӮ

UDDI з”ЁжқҘз®ЎзҗҶпјҢеҲҶеҸ‘пјҢжҹҘиҜў WebServiceгҖӮ

дәҢгҖҒSOAP з®Җиҝ°

SOAP ж¶ҲжҒҜз»„жҲҗйғЁеҲҶпјҡжүҖжңүзҡ„ SOAP ж¶ҲжҒҜйғҪдҪҝз”Ё XML зј–з ҒпјҢдёҖжқЎ SOAP ж¶ҲжҒҜе°ұжҳҜдёҖдёӘжҷ®йҖҡзҡ„ XML ж–ҮжЎЈпјҢиҜҘж–ҮжЎЈеҢ…жӢ¬дёӢеҲ—е…ғзҙ гҖӮ

SOAP ж¶ҲжҒҜж јејҸдҫӢеӯҗ

дёүгҖҒSOAP жҺҘеҸЈжјҸжҙһ

SOAP еһӢ Web Service жјҸжҙһзұ»еһӢдёҺ Web жјҸжҙһзұ»еһӢеҢәеҲ«дёҚеӨӘпјҢеҸӘжҳҜиҜ·жұӮзҡ„ payload жһ„йҖ йңҖиҰҒж»Ўи¶ідёҖдәӣж јејҸиҰҒжұӮпјҢе…·дҪ“иҰҒзңӢWeb ServiceжңҚеҠЎз«Ҝд»Јз Ғзј–еҶҷжғ…еҶөгҖӮжјҸжҙһзұ»еһӢжҜ”еҰӮе‘Ҫд»ӨжіЁе…ҘгҖҒSQLжіЁе…ҘгҖҒXSSгҖҒXXEгҖҒXPathжіЁе…ҘгҖҒDoSгҖҒйҖ»иҫ‘жјҸжҙһгҖҒдҝЎжҒҜжі„йңІвҖҰвҖҰзӯүзӯүгҖӮ

еҰӮдёӢдёәдёҖдёӘе‘ҳе·Ҙиө„ж–ҷжҹҘиҜўеҠҹиғҪпјҢе…¶дёӯ<wkno>string</wkno> <dept>string</dept> <memo>string</memo> <status>string</status> <sex>string</sex> <agefilter>string</agefilter> <name>string</name> <token>string</token> дёәз”ЁжҲ·иҫ“е…Ҙзҡ„ж•°жҚ®,еҸҜдҪңдёәжөӢиҜ•зӮ№иҝӣиЎҢжјҸжҙһжөӢиҜ•гҖӮ

дҫӢеҰӮеңЁзҪ‘йЎөдёҠеҸ‘зҺ°еӯҳеңЁ SOAP жҺҘеҸЈпјҢжҲ‘д»¬е°ұеҸҜд»ҘеҜ№е…¶иҫ“е…ҘзӮ№иҝӣиЎҢжөӢиҜ•гҖӮ

жөӢиҜ•зҡ„ж—¶еҖҷжҲ‘д»¬дјҡйҒҮеҲ° xxx?WSDL йЎөйқўпјҢиҝҷж—¶еҖҷжҲ‘д»¬е°ұеҸҜд»ҘеҲ©з”Ё Burp жҸ’д»¶ Wsdler жҸҗеҸ–жҺҘеҸЈиҝӣиЎҢжөӢиҜ•гҖӮ

burp е•Ҷеә—е®үиЈ… Wsdler жҸ’д»¶

жүҫеҲ° asmx?WSDL йЎөйқўпјҢжҠ“еҸ–ж•°жҚ®еҢ…зӣҙжҺҘеҸій”® Extensions еӨ„жҸҗеҸ–жҺҘеҸЈ

жҲҗеҠҹжҸҗеҸ–ж•°жҚ®жҺҘеҸЈпјҢжҺҘдёӢжқҘе°ұеҸҜд»ҘиҝӣиЎҢжјҸжҙһжөӢиҜ•

еӣӣгҖҒжҗңзҙў Web Service жңҚеҠЎ

Google жҢҮе®ҡжҗңзҙў Web Service жңҚеҠЎ,еҰӮдёӢжүҖзӨәпјҢжҢҮе®ҡз«ҷзӮ№ж—¶еҸҜеңЁеүҚйқўж·»еҠ  site:x.x.x.x

дә”гҖҒе°Ҹз»“

йҖҡиҝҮдәҶи§ЈеӯҰд№ еҗҺеҸ‘зҺ°пјҢsoap жҺҘеҸЈжөӢиҜ•дёҺеёёи§„зҡ„жөӢиҜ•еҢәеҲ«дёҚеӨ§пјҢдё»иҰҒеҢәеҲ«е°ұжҳҜи°ғз”Ёзҡ„жҺҘеҸЈдҪҝз”Ё XML ж јејҸпјҢжөӢиҜ•ж—¶йңҖиҰҒжһ„йҖ зӣёе…іиЎЁеҚ•иҝӣиЎҢжөӢиҜ•гҖӮ
